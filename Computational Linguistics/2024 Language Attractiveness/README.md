# 🗣️ What Makes a Language Sound Attractive?

## Research Question

What linguistic and phonetic features influence **French speakers' perceptions of language attractiveness**?

This project explores how specific features — such as **sound patterns and phonetic diversity** — shape preferences for foreign languages.

It is inspired by the paper _“Do some languages sound more beautiful than others?”_ (Anikin et al.), with a specific focus on **French speakers**.

---

## 🧠 Hypotheses

The project tests the following predictions:

- **P1 — Linguistic Comfort Zone**
  Languages belonging to the same linguistic subfamily as the speaker (here, **Romance languages**) are perceived as more attractive.

- **P2 — Phonetic Diversity**
  Languages with a **smaller variety of vowels** are perceived as less attractive.

---

## 📊 Data

The dataset consists of **survey responses I designed**:

- attractiveness ratings for different languages
- responses from **French speakers**
- linguistic features associated with each language

---

## Modeling Approach

### Linear Regression (OLS)

A **linear regression model (OLS)** was used to explain attractiveness scores.

---

### Statistical Testing

To test the hypotheses:

- **t-tests** were used to compare:
  - Romance vs non-Romance languages (P1)
  - languages with different levels of vowel diversity (P2)

- significance tests on regression coefficients

---

## 📊 Key Results

- French speakers tend to prefer languages closer to their **linguistic background**
- phonetic features (such as vowel diversity) influence perception
- attractiveness judgments show **systematic patterns**, not purely subjective
